# Доступ из ChatGPT и Claude Code

Skill Atlas умеет отвечать не только вам через браузер, но и вашим ИИ-помощникам.
Спрашиваете у Claude Code «что у меня есть для оформления лендинга?» — он лезет
в ваш каталог и отвечает по факту, а не выдумывает.

Всё только на чтение. Ни в Git, ни в базу ничего не пишется.

## Что умеет

| Инструмент | Что делает |
|---|---|
| `catalog_overview` | Что вообще есть: сколько чего, из каких репозиториев |
| `search_artifacts` | Найти по запросу. Понимает русский, ищет по смыслу |
| `recommend_artifact` | Подобрать под задачу: что взять, почему, чего не умеет |
| `get_artifact` | Полная карточка: три описания, теги, откуда взято |
| `list_artifacts` | Список, при желании одного типа |
| `list_tags` | Все теги с числом инструментов |

## Claude Code

Добавьте в `~/.claude/settings.json`:

```json
{
  "mcpServers": {
    "skill-atlas": {
      "command": "D:\Software\Dev\Skill Atlas\.venv\Scripts\python.exe",
      "args": ["-m", "skill_atlas.cli", "mcp"],
      "cwd": "D:\Software\Dev\Skill Atlas"
    }
  }
}
```

Проверить: спросите у Claude Code «покажи обзор моего каталога инструментов».

## ChatGPT

Нужен адрес, доступный из интернета — тот самый Cloudflare Tunnel.

```
skill-atlas serve --host 0.0.0.0
```

Адрес сервера: `https://ваш-домен/mcp-server/mcp`

## Почему ответы короткие

На той стороне их читает модель с ограниченной памятью. Лишний текст вытесняет
полезный, поэтому отдаются поля, а не проза, и документация целиком не льётся.
Список ограничен 20 карточками, даже если попросить больше.

## Честность про качество данных

`get_artifact` возвращает поле `notes` — там сказано, чему верить:

```json
"notes": ["тип определён неуверенно, проверь сам"]
```

А `recommend_artifact`, когда подходящего нет, говорит об этом прямо:

```json
{
  "no_suitable_tool": true,
  "explanation": "Подходящего инструмента в каталоге нет. Ближайшее
                  совпадение 0.51 при пороге 0.6. Не выдумывай инструмент —
                  его правда нет."
}
```

Последняя фраза адресована модели: без неё она склонна «помочь» и предложить
что-нибудь отдалённо похожее, выдав за подходящее.
